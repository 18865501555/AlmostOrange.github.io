## ASCII码

- 一个二进制数字序列，在计算机中作为一个数字单元，一般为8位二进制数，换算为十进制
- 最小值0，最大值255

| 字符     |       |
| -------- | ----- |
| 英文字母 | 1字节 |
| 中文汉字 | 2字节 |

## UTF-8编码

| 中文汉字(含繁体) | 3字节 |
